|     类型     |   符号    | 占位符 | 字节 |
| :----------: | :-------: | :----: | :--: |
|     字符     |   char    |   %c   |  1   |
|   字符串型   |           |   %s   |      |

|    短整型    |   short   |  %hd   |  2   |

## 储存类

|  存储类  |                         描述                         |
| :------: | :--------------------------------------------------: |
| register |    用于定义存储在寄存器中而不是 RAM 中的局部变量     |
|  static  | 存储类指示编译器在程序的生命周期内保持局部变量的存在 |
|  extern  |         定义在其他文件中声明的全局变量或函数         |
