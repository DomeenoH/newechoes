---
title: 常用全局属性
date: 2024-06-06T23:51:09Z
tags: []
---



1. id
   - 标签唯一标识
   - 不能在以下元素使用head,html,meta,script,style,title
2. class
   - 设置类名
   - 可以通过css给一个类设置样式
3. style
   - 给css设置样式
4. dir
   - 内容的方向,值:ltr/rtl
   - 不能和id不能的元素使用一样
5. title
   - 给标签一个提示
6. lang
   - 给标签指定语言
   - 不能和id不能的元素使用一样
