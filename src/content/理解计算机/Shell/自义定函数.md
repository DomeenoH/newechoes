## 方法一：使用 function 关键字

```shell
function function_name {
    # 函数体
    # 可以包含多行命令
    echo "This is a custom function."
}
```

## 方法二：直接定义函数名称

```shell
function_name() {
    # 函数体
    # 可以包含多行命令
    echo "This is a custom function."
}
```
