### 局部变量:以@开头,先声明,再赋值

```sql
declare @变量名 类型
set @变量名 = 值
```

或

```sql
select @变量名=值
```

> 变量只能接收一个值
> set：赋值变量指定的值（只能插入一条）
> select:一般用于表中查询出的数据赋值给变量(如果有多条,把最后一行赋值给变量)
