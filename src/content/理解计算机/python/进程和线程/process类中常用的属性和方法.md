title: process类中常用的属性和方法
date: 2024-06-06T23:51:47Z
tags: []
---


| 描述                               | 属性     |
| ---------------------------------- | -------- |
| 当前进程的实例别名,默认为Process-N | ​`name`​ |
| 当前进程对象的PID值                | ​`pid`​  |

| 描述                                                      | 方法              |
| --------------------------------------------------------- | ----------------- |
| 进程是否执行完,没执行完结果为True,否则为False             | ​`is_alive()`​    |
| 等待结束或等待timeout秒                                   | ​`join(timeout)`​ |
| 启动进程                                                  | ​`start()`​       |
| 如果没有指定target参数,则启动进程后,会调用父类中的run方法 | ​`run()`​         |
| 强制终止进程                                              | ​`terminate()`​   |
