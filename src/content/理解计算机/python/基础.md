## 输出

​`print("")`​

- Python 允许使用单引号、双引号、三重引号（单引号或双引号）来表示字符串
- `end=''`输出以空格结尾，而不是默认的换行符
- `sep=","`指定了逗号作为分隔符。
  输入到文件: `print("文字",file=文件名)`
  完整结构:`print(value,,sep='',end='\n',file='')`
- `flush=True` 不会立即输出
